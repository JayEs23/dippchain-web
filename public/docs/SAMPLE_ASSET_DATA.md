## üí° Pro Tips

### For Different File Types

**Images:**
- JPG/PNG for photos
- PNG for graphics with transparency
- Recommended size: 1920x1080 or larger

**Videos:**
- MP4 (H.264) most compatible
- Keep under 50MB for testing
- 1080p or 720p resolution

**Audio:**
- MP3 or WAV formats
- 320kbps for high quality
- Include cover art in metadata

**Documents:**
- PDF for documents
- Add preview thumbnail
- Include page count in description
